Semantic segmentation for change detection

Semantic segmentation for change detection

  The work of company is a change detection project using semantic segmentation, which can identify the changes in buildings of Seoul Metropolitan by year using satellite photography. In order to train the learning, I needed to understand the characteristics of the data that I want to train, so I did labeling and preprocessing the data. The train method was to concatenate the 2015 aerial image and the 2017 aerial image, and setting the changed part of the two years as the mask.

Multi-label classification at face image

Multi-label classification at face image

  The classes are divided into five classes: skin color, hair color, glasses presence, glasses type, and eyebrow type. Since the classifier of the trained model will work in the mobile environment, I constructed the model using Mobilenet ver2.

  I also applied DANN (Domain adversarial neural network) as a side project. The reason I wanted to apply DANN was that the company used the Unity engine to create a lot of face photos by combining the various cases of each class. Since the company didn’t have many real human face photos and it's hard to label them manually, they have created a tool that can be set the labels as desired. However, the image created by the Unity engine is a game avatar-like image, which is very different from the real face image.

  Therefore, to overcome this problem, we tried to make confuse the domain by using DANN. However, the reason for the large domain difference between real and synthetic images, The training did not overcome the problem even when using DANN.

  Due to the limited internship period, the project could not be further progressed. I think it could be overcome to change the domain of the synthetic image to real image domain by using the cycleGAN method.

Computer vision automatic door

Computer vision automatic door

  This project was to create a product that can be easily converted from hinged door to automatic door. Existing doors are difficult to use for people who are under heavy or need help, such as those with heavy loads, wheelchairs, or children with limited strength. Therefore, we wanted to build a product that would provide the power to open these hinged doors. The goal was not to change the door itself to the automatic doors, but to create the convenience machine of easy installation and the reasonable price.

  In addition, there was a simple way to open the door through human body detection sensor, RFID, NFC card recognition, etc. However, we chose more complex way to recognize people which is computer vision technology. We used a haar cascade and extracted features based on the per-pixel brightness of the face image. Faces are detected by drawing a bounding box on the face using a webcam and opencv.

  The hardware used solid works to design the arm structure opening the body of the door and moving the motor.

  I designed the PID control in Arduino to control the movement of the motor. In communication with the Raspberry Pi, let the motor open the door according to the facial recognition information from the webcam.

Dormitory Public laundry monitoring system

Dormitory Public laundry monitoring system

  My team have developed Dormitory Public laundry monitoring system at DGIST making club.

  Most of DGIST students lives in dormitory. Students have to use the washing machine and dryer in the laundry room located on one floor of the dormitory. However, many people use a washing machine and a dryer, so when students go to use laundry, the washing machine is all in use, so they have to wait a long time.

  Also, if they keep the washing machine running and do other things, it is difficult to know the remaining time for the washing machine they have returned. Therefore, I wanted to develop a service to check the operating state of washing machines and dryers to help students use the laundry room more conveniently and smartly.

  Using computer vision technology, we can get whether the washing machine's LEDs were working. Therefore, we obtained several LED images in the dormitory laundry room, and divided the cases whether the machine is working or not. I trained LED images using LBP cascade method.

  The result of the training was used to classify the operation status by receiving the information from the laundry room in real time using the webcam. The hardware for installing the webcam and the Raspberry Pi was manufactured in Solidworks. Raspberry Pi checked the operating status from the laundry room by using the trained model and the webcam, and sending it to the server. The web front-end which is client can get the information from the server and show information to users.
